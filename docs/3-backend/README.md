# 🤖 3: Backend

## Learning Objectives

1. Build backend applications that handle requests, business logic, and data persistence
2. Create RESTful APIs with Java and Spring Boot
3. Work with relational databases using PostgreSQL and SQL
4. Use Spring Data JPA / Hibernate to map Java objects to database tables
5. Add authentication and authorization with Spring Security
6. Test and deploy backend services that support full-stack applications

## Introduction

![We will now build the backend portion of our app architecture. Source: Rocket Academy](<../.gitbook/assets/3 - Backend.png>)

In Module 3: Backend we will build backend services with Java and Spring Boot that store, retrieve, and secure data for our frontends. Instead of the older Express.js and Sequelize flow, this module focuses on Java-based APIs, PostgreSQL databases, Spring Data JPA, and Spring Security so the repo matches the current curriculum.
