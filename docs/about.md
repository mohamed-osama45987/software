# About

This site hosts the assignments and learning materials for the Tamkeen Software Engineering programme, delivered by [Skills Union](https://skillsunion.com).

## How to use this site

- Browse lessons using the navigation links
- Each lesson has a companion **Assignments** page with quizzes and coding tasks
- Click **Show hint** on any question to reveal a pointer to the relevant lesson section — hints never give away the answer
- Submit your completed work on [Skills Union](https://skillsu.com/member/assessment)

## Curriculum

The programme covers:

- **Module 0** — JavaScript foundations (Promises, async/await)
- **Module 2** — Full stack web development (React, AJAX, Firebase)
- **Module 3** — Backend development (Java, Spring Boot, PostgreSQL)
- **Module 4** — DevOps (CI/CD, containers, security, monitoring)
- **Algorithms** — Data structures and search algorithms
- **React Native** — Mobile app development
