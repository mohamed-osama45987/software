## How to use these

Skim the lesson first, then attempt the assessment closed-book. It's normal to look up syntax — these are not closed-book exams. When stuck, click **Show hint** to get a pointer to the relevant lesson section; hints never name the correct answer. Submit completed work on [Skills Union](https://skillsu.com/member/assessment).
